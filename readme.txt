== Description ==
This plugin allows creation of XML sitemaps for submission to Google. It adds a small widget box in the admin panel when you edit or create posts and pages, that allows exclusion from the sitemap file.
A special thanks to bestwebsoft for their awesome plugin called google-sitemap-plugin


== Installation ==
1. Upload all the plugin files to a folder called iXML in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the `settings` link to edit the plugin definitions
